# Geo Assets

`india-states.geojson` contains India state/union-territory boundaries filtered from Natural Earth Admin 1 states/provinces GeoJSON.

Source used during development:

`https://raw.githubusercontent.com/nvkelso/natural-earth-vector/master/geojson/ne_10m_admin_1_states_provinces.geojson`

This file is only for the PAN India state territory layer. POP/service-area territory polygons are not available yet; the current POP layer uses centroid markers from site latitude/longitude.
